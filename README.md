# "Who got banned", a small simple name matcher for League of Legends

Developed using node 22 (requires 18, for global fetch).

## Playing around

```bash
pnpm i
pnpm run dev
```

## Building

```bash
pnpm run build
pnpm run start
```

## Docker

```bash
docker build .
```
